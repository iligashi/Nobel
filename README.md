# Nobel Venues

Luxury event venue website for Nobel Venues, located in Drenas, Kosova. A modern, elegant single-page application showcasing premium event hosting spaces with bilingual support (English & Albanian).

## Tech Stack

- **React 19** — UI framework
- **Tailwind CSS 3** — Utility-first styling
- **Framer Motion** — Animations and transitions
- **React Router** — Client-side routing
- **React Icons** — Icon library

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

### Production Build

```bash
npm run build
```

Outputs optimized files to the `build/` folder.

## Features

- Bilingual support (English / Albanian)
- Responsive design for all devices
- Smooth scroll animations
- Multiple venue showcases (Salla 1, Salla 2, Salla 3)
- Contact section
